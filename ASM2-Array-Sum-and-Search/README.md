# ASM2 Array Sum and Search

Write small array routines in assembly and trace every iteration.

## Goals

- practice loop structure with labels and conditional jumps
- load array elements through scaled addressing
- return aggregate and search results through registers

## Starter Expectations

- finish the search routine
- explain the exit condition for each loop
- annotate which address is visited on each pass

## Solution Highlights

- sums a signed integer array
- searches for the first value that meets a threshold
- keeps the loop state readable in a debugger
