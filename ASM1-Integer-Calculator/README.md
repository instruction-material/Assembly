# ASM1 Integer Calculator

Build a tiny integer calculator in `x86-64 assembly`.

## Goals

- move values through argument and return registers deliberately
- practice arithmetic and logic instructions
- trace operator-driven dispatch through comparisons and jumps

## Starter Expectations

- extend the operator table beyond the already working basics
- explain which registers hold the live operands at each branch
- compare the assembly flow to the equivalent C expression logic

## Solution Highlights

- supports arithmetic and bitwise operators
- shows a compact branch-based dispatch routine
- keeps the function small enough for full register tracing
